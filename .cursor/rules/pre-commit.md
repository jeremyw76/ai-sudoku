# Mandatory pre-commit rules

Before creating any commit:
1. Copy `./AGENT_LOGS.md.template` to `./AGENT_LOGS.md`
2. Append the contents of all log files from `./docs/agent-logs` to `./AGENT_LOGS.md`.
3. Stage `./AGENT_LOGS.md` for commit.
4. Delete all log files from `./docs/agent-logs`.

Before appending to any commit:
1. Append the contents of all log files from `./docs/agent-logs` to `./AGENT_LOGS.md`
2. Stage `./AGENT_LOGS.md` for commit.
3. Delete all log files from `./docs/agent-logs`.
